### Requerimientos📋

_Es necesario instalar las dependencias figuradas en el package.json_

```
npm i 
```
## Ejecucion ⌨️
_En la consola escriba el siguiente comando:_
```
- npm start
```
_Luego vaya al navegador a la siguiente ruta_
```
localhost: 3000
```
